# AI 模拟面试系统 - 项目概述

## 📋 项目简介

这是一个基于 AI 大模型的智能模拟面试系统，支持语音对话、简历分析、多轮追问和评估报告生成。系统分为 **Python 后端** 和 **Vue3 前端** 两部分，可独立部署或组合使用。

### 核心能力

- 🎤 **语音交互** - 支持麦克风输入、语音播报，模拟真实电话面试体验
- 📄 **简历分析** - 自动解析简历，提取技术栈和项目经历
- 🎯 **智能追问** - 根据回答内容进行深度追问，还原真实面试压迫感
- 📊 **评估报告** - 面试结束后生成多维度评分和改进建议
- 💾 **数据持久化** - SQLite 数据库存储用户、面试记录和历史

## 🗄️ 数据库设计（SQLite）

### 数据库配置

| 配置项 | 值 |
|--------|-----|
| 数据库类型 | SQLite |
| 存储位置 | `data/interview.db` |
| 并发模式 | WAL (Write-Ahead Logging) |
| 外键约束 | 启用 |
| 表数量 | 6 |

### 数据表结构

#### 1. users（用户表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| phone | VARCHAR(20) | 手机号，唯一 |
| email | VARCHAR(255) | 邮箱 |
| nickname | VARCHAR(100) | 昵称 |
| avatar_url | VARCHAR(500) | 头像URL |
| settings | JSON | 用户设置 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    phone VARCHAR(20) UNIQUE,
    email VARCHAR(255),
    nickname VARCHAR(100),
    avatar_url VARCHAR(500),
    settings JSON DEFAULT '{}',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 2. resumes（简历表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| user_id | INTEGER | 外键，用户ID |
| name | VARCHAR(100) | 姓名 |
| phone | VARCHAR(20) | 电话 |
| email | VARCHAR(255) | 邮箱 |
| education | TEXT | 教育经历 |
| work_experience | TEXT | 工作经历 |
| skills | JSON | 技能列表 |
| projects | JSON | 项目经历 |
| raw_text | TEXT | 原始文本 |
| file_url | VARCHAR(500) | 文件URL |
| is_default | BOOLEAN | 是否默认简历 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

```sql
CREATE TABLE resumes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    name VARCHAR(100),
    phone VARCHAR(20),
    email VARCHAR(255),
    education TEXT,
    work_experience TEXT,
    skills JSON DEFAULT '[]',
    projects JSON DEFAULT '[]',
    raw_text TEXT,
    file_url VARCHAR(500),
    is_default BOOLEAN DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

#### 3. interviews（面试记录表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| user_id | INTEGER | 外键，用户ID |
| resume_id | INTEGER | 外键，简历ID |
| position | VARCHAR(100) | 应聘岗位 |
| level | VARCHAR(20) | 职级（初级/中级/高级） |
| mode | VARCHAR(20) | 面试模式（语音/文字） |
| status | VARCHAR(20) | 状态（pending/ongoing/completed） |
| duration | INTEGER | 面试时长（秒） |
| started_at | TIMESTAMP | 开始时间 |
| ended_at | TIMESTAMP | 结束时间 |
| created_at | TIMESTAMP | 创建时间 |

```sql
CREATE TABLE interviews (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    resume_id INTEGER,
    position VARCHAR(100) NOT NULL,
    level VARCHAR(20) DEFAULT '中级',
    mode VARCHAR(20) DEFAULT '语音',
    status VARCHAR(20) DEFAULT 'pending',
    duration INTEGER DEFAULT 0,
    started_at TIMESTAMP,
    ended_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (resume_id) REFERENCES resumes(id) ON DELETE SET NULL
);
```

#### 4. messages（对话消息表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| interview_id | INTEGER | 外键，面试ID |
| role | VARCHAR(20) | 角色（user/assistant） |
| content | TEXT | 消息内容 |
| audio_url | VARCHAR(500) | 语音URL |
| audio_duration | FLOAT | 语音时长 |
| created_at | TIMESTAMP | 创建时间 |

```sql
CREATE TABLE messages (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    interview_id INTEGER NOT NULL,
    role VARCHAR(20) NOT NULL,
    content TEXT,
    audio_url VARCHAR(500),
    audio_duration FLOAT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (interview_id) REFERENCES interviews(id) ON DELETE CASCADE
);
```

#### 5. reports（评估报告表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| interview_id | INTEGER | 外键，面试ID，唯一 |
| user_id | INTEGER | 外键，用户ID |
| overall_score | DECIMAL(3,1) | 综合评分 |
| technical_score | DECIMAL(3,1) | 技术能力评分 |
| project_score | DECIMAL(3,1) | 项目经验评分 |
| communication_score | DECIMAL(3,1) | 沟通表达评分 |
| logic_score | DECIMAL(3,1) | 逻辑思维评分 |
| pressure_score | DECIMAL(3,1) | 抗压能力评分 |
| strengths | TEXT | 优点 |
| weaknesses | TEXT | 缺点 |
| suggestions | TEXT | 改进建议 |
| feedback | JSON | 详细反馈 |
| created_at | TIMESTAMP | 创建时间 |

```sql
CREATE TABLE reports (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    interview_id INTEGER NOT NULL UNIQUE,
    user_id INTEGER NOT NULL,
    overall_score DECIMAL(3,1) DEFAULT 0,
    technical_score DECIMAL(3,1) DEFAULT 0,
    project_score DECIMAL(3,1) DEFAULT 0,
    communication_score DECIMAL(3,1) DEFAULT 0,
    logic_score DECIMAL(3,1) DEFAULT 0,
    pressure_score DECIMAL(3,1) DEFAULT 0,
    strengths TEXT,
    weaknesses TEXT,
    suggestions TEXT,
    feedback JSON DEFAULT '{}',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (interview_id) REFERENCES interviews(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

#### 6. favorited_questions（收藏问题表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| user_id | INTEGER | 外键，用户ID |
| interview_id | INTEGER | 外键，面试ID |
| question | TEXT | 问题内容 |
| answer | TEXT | 参考回答 |
| notes | TEXT | 备注 |
| created_at | TIMESTAMP | 创建时间 |

```sql
CREATE TABLE favorited_questions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    interview_id INTEGER,
    question TEXT NOT NULL,
    answer TEXT,
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (interview_id) REFERENCES interviews(id) ON DELETE SET NULL
);
```

## 🚀 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- npm 或 yarn

### 1. 启动后端服务

```bash
# 进入项目目录
cd /workspace/projects

# 安装 Python 依赖（已在 .venv 中）
source .venv/bin/activate

# 启动 HTTP 服务（自动初始化数据库）
python src/main.py -m http -p 5000

# 或使用脚本启动
bash scripts/http_run.sh

# 访问健康检查
curl http://localhost:5000/health
```

### 2. 启动前端开发服务器

```bash
# 进入 Vue3 项目目录
cd vue3-interview

# 安装依赖
npm install

# 启动开发服务器（默认端口 5173）
npm run dev

# 访问 http://localhost:5173
```

### 3. 生产部署

```bash
# 构建前端
cd vue3-interview
npm run build

# 部署 dist/ 目录到 Nginx

# 后端服务（推荐使用 systemd）
sudo systemctl start interview-backend
```
