# 模拟面试 Agent 架构设计文档

## 📋 概述

这是一个基于 **LangGraph + LangChain** 构建的智能模拟面试系统，通过模块化设计实现了语音交互、简历分析、连环追问和评估报告等功能。

## 🔧 核心模块详解

### 1. Agent 核心 (`src/agents/agent.py`)

```python
# 伪代码结构
def build_agent(ctx=None):
    # 1. 加载 LLM 配置
    cfg = load_config("config/agent_llm_config.json")
    
    # 2. 初始化 LLM (DeepSeek)
    llm = ChatOpenAI(
        model="deepseek-chat",
        api_key=DEEPSEEK_API_KEY,
        streaming=True
    )
    
    # 3. 注册工具
    tools = [
        text_to_speech,       # 语音输出
        speech_to_text,       # 语音输入
        analyze_resume_text,   # 简历分析
        generate_interview_report,  # 报告生成
        # ... 更多工具
    ]
    
    # 4. 构建 Agent (LangGraph)
    return create_agent(
        model=llm,
        system_prompt=cfg.get("sp"),  # 面试官角色
        tools=tools,
        checkpointer=get_memory_saver(),  # 对话记忆
        state_schema=MessagesState,
    )
```

**关键点**：
- `create_agent()` 是 LangGraph 的高层 API
- `checkpointer` 实现对话历史持久化
- `state_schema=MessagesState` 定义状态结构

@tool
def text_to_speech(text: str, voice: str = "female") -> str:
    """文字转语音"""
    client = TTSClient()
    audio_url, size = client.synthesize(
        text=text,
        speaker=VOICE_OPTIONS[voice]
    )
    return f"success|{audio_url}|{size}"

**关键代码**：
```python
# 使用滑动窗口保留最近 40 条消息
MAX_MESSAGES = 40

def _windowed_messages(old, new):
    """滑动窗口: 只保留最近 MAX_MESSAGES 条消息"""
    return add_messages(old, new)[-MAX_MESSAGES:]

class AgentState(MessagesState):
    messages: Annotated[list[AnyMessage], _windowed_messages]

# 在 create_agent 时传入
checkpointer=get_memory_saver()
```
